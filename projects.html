Display Profile Switcher - GNOME Extension

I use different screen set-ups with my laptop and was annoyed by the inconvenience of switching between different set-ups. So I wrote this GNOME extension, that (for me) does the trick: Display Profile Switcher.

I did this on a sunday because my current set-up was really bugging me. I am curious if this project is helpful to anybody else. Once I've tested it more, I plan to also release it to extensions.gnome.org (no idea how this works, though).

There are also more features I would like to add, but I am just putting this out there for now.

You can read all about my motivation for this project, as well as a full guide on how to install and configure it on the projects github page

Install LineageOS with microG and root on Galaxy Pixel 3a

For my smartphone I have some requirements:

  • No bloatware
  • No proprietary service I have no control over
  • Possiblity to use a hostfile to block trackers, malware, etc (imho the only necessary virus protection, but then again, what to I know - please dont hack me)

So I wanted to have LineageOS with microG (for access to GCM, which need for banking) and root (to change my hostfile). Unfortunately that seems not to be so easy, even if you have a easily unlockable phone like the Pixel 3a. But after some hours I had put together a number of snippets that resembled a "solution" to my problem.

Please not that I did not write most of that code and do not know if all of that is safe to use. But it works (for me).
